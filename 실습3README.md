# 📖 그림 이야기 생성기

> 실습문제 3 — 그림과 감정을 기반으로 짧은 이야기를 생성하는 웹 애플리케이션

---

## 🖥️ 실행 방법

HTML 파일을 브라우저에서 바로 열면 됩니다. 별도 서버 설치가 필요 없습니다.

```bash
# 터미널에서 실행 (Mac)
open 실습3.html

# 특정 브라우저로 열기
open -a "Google Chrome" 실습3.html
open -a "Safari" 실습3.html
```

또는 **Finder에서 `실습3.html` 파일을 더블클릭**하세요.

---

## ✅ 사전 요구사항

- **OpenAI API Key** (gpt-4o 사용 권한 필요)
  - [https://platform.openai.com/api-keys](https://platform.openai.com/api-keys) 에서 발급
- 인터넷 연결 (OpenAI API 호출 및 Google Fonts 로드)
- 최신 버전 브라우저 권장 (Chrome, Safari, Firefox, Edge)

---

## 📋 사용 방법

### 1단계 — API Key 입력
- 페이지 상단의 **🔑 OpenAI API Key** 입력란에 `sk-...` 형식의 API 키를 입력합니다.

### 2단계 — 감정 선택
- **감정 선택** 섹션에서 이야기의 분위기를 결정할 감정 버튼을 클릭합니다.
- 선택된 감정에 따라 페이지 배경색과 글씨체가 자동으로 변경됩니다.
- 총 **18가지 감정** 중 하나를 선택할 수 있습니다.

| 감정 | 감정 | 감정 |
|------|------|------|
| 😊 행복한 | 😢 슬픈 | 😰 긴장감 있는 |
| 🥰 설레는 | 🌌 신비로운 | 🌸 따뜻한 |
| 🍂 슬프고 아름다운 | 😄 유쾌한 | 😱 무서운 |
| ✨ 환상적인 | 🌙 외로운 | 🕊️ 그리운 |
| 🦁 용감한 | 🌿 평화로운 | 🌊 몽환적인 |
| ⚡ 웅장한 | 🐣 귀여운 | 🎢 짜릿한 |

### 3단계 — 그림 업로드
- **그림 업로드** 섹션의 점선 영역을 **클릭**하거나 이미지를 **드래그 앤 드롭**합니다.
- 여러 장의 이미지를 업로드할 수 있으며, 업로드 순서가 이야기 순서가 됩니다.
- 지원 형식: **JPG, PNG, WEBP**
- 이미지 썸네일의 **✕ 버튼**으로 개별 삭제, 숫자 배지로 순서 확인 가능

### 4단계 — 이야기 생성
- API Key, 감정, 이미지가 모두 준비되면 **✨ 이야기 생성하기** 버튼이 활성화됩니다.
- 버튼을 클릭하면 OpenAI GPT-4o가 그림을 분석하고 이야기를 생성합니다.
- 생성에는 약 **5~15초**가 소요됩니다.

### 5단계 — 결과 확인
- 생성된 이야기가 하단 **📜 생성된 이야기** 섹션에 표시됩니다.
- 선택한 감정 태그와 이야기 글자 수가 함께 표시됩니다.
- 새로운 감정이나 이미지로 다시 생성하려면 위 단계를 반복합니다.

## 🌈 감정별 동적 테마

감정을 선택하면 페이지 전체가 해당 감정의 분위기에 맞게 변합니다.

| 항목 | 변경 내용 |
|------|-----------|
| **배경** | 감정에 맞는 색상 그라데이션으로 전환 (0.7초 애니메이션) |
| **테두리·강조색** | 섹션 카드, 버튼, 입력창 색상이 테마에 맞게 변경 |
| **글씨체** | 서정적 감정(슬픈, 그리운 등)은 명조체+이탤릭, 활동적 감정은 고딕체 |

- API Key를 소스 코드에 직접 입력하거나 공개 저장소에 업로드하지 마세요.
- 이 파일은 로컬 실행용입니다. 웹 서버에 배포 시 API Key 관리에 주의하세요.

## ❓ 자주 묻는 문제

| 문제 | 해결 방법 |
|------|-----------|
| 버튼이 비활성화 상태 | API Key 입력 + 감정 선택 + 이미지 1장 이상 업로드 모두 필요 |
| `API 오류: 401` | API Key가 잘못되었거나 만료됨. 키를 재확인하세요 |
| `API 오류: 429` | API 사용량 한도 초과. 잠시 후 다시 시도하세요 |
| 이미지가 안 올라감 | JPG/PNG/WEBP 형식인지 확인, 파일 크기가 너무 크면 압축 필요 |
| 글씨가 깨져 보임 | 인터넷 연결 확인 (Google Fonts 로드 필요) |
